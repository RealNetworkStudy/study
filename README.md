## 주차별 주제
### [1주차]
 브라우저에 url을 입력하면, 브라우저에서 url(웹 서버)에 요청을 보낸다. 동작 흐름에 따라서 조사하기! 

 > IP, DNS, Socket, Protocol stack

 상세 과정 
 1. HTTP request 메시지를 작성한다.
 2. 웹 서버(url)의 IP 주소를 DNS 서버에 조회한다
 3. 프로토콜 스택에 메시지 송신을 의뢰한다.
